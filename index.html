Temas GKG dominantes

Categorías de la taxonomía de GDELT más frecuentes en estas menciones.

Metodología: se descargan los archivos GKG 2.1 públicos de data.gdeltproject.org/gdeltv2/ (sin necesidad de API key), se filtran las filas cuyo campo V2Locations contiene el código de país MX, y se calcula el promedio de V2Tone por ventana de 15 minutos y por fuente. Pipeline en Python, reutilizable para cualquier país o periodo histórico vía BigQuery.

Para producción: este snapshot se puede convertir en un monitor verdaderamente en vivo programando el script cada 15 min (cron / GitHub Actions) y sirviendo el JSON resultante a este dashboard, o consultando directamente la tabla gdelt-bq:gdeltv2.gkg en BigQuery para históricos y comparativos multi-país.
